This project is based on a Convolutional Neural Network (CNN) model used for waste segregation. The main aim of the project is to automatically identify and classify different types of waste such as plastic, paper, metal, glass, and organic waste using image processing and deep learning techniques.

In this system, images of waste materials are given as input to the CNN model. The model is trained using a dataset containing different categories of waste images. During training, the CNN learns important features like shape, color, and texture of each waste type. After training, the model can accurately predict the category of a new waste image.

The project helps in improving waste management by reducing manual sorting effort, saving time, and increasing recycling efficiency. It can be used in smart dustbins, recycling plants, and automated waste management systems to support cleaner and smarter environmental practices.
